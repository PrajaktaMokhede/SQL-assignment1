# Retail Store Analytics System

## Project Overview

The Retail Store Analytics System is a SQL-based database project designed to analyze customer purchases, product performance, sales trends, payment methods, and city-wise business performance.

This project demonstrates the use of SQL concepts ranging from basic queries to advanced reporting techniques using a retail sales dataset.

## Objective

The objective of this project is to practice and implement:

- Database Creation
- Table Creation
- Data Insertion
- Data Retrieval
- Filtering Records
- Sorting Data
- Aggregate Functions
- String Functions
- Numeric Functions
- Date Functions
- UPDATE Operations
- DELETE Operations
- GROUP BY Clause
- HAVING Clause
- Business Report Generation
